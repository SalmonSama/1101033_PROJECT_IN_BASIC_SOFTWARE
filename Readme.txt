เว็บไซต์ร้านข้าวขาหมูลุงรี

wordpress version: 6.4.3
xampp version : 8.2.12

Installation:

- ติดตั้ง xampp 
- copy folder khamulungree ไปวางไว้ที่ ~\xampp\htdocs\
- สร้างฐานข้อมูลผ่าน http://localhost/phpmyadmin/
- ตั้งชื่อฐานข้อมูลเป็น sql_khamulungree 
- กำหนด collation ของฐานข้อมูลให้เป็น utf8_general_ci
- นำเข้าฐานข้อมูล
- เข้าเว็บไซต์ผ่าน http://localhost/khamulungree/
